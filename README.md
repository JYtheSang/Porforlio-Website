
# Jie Yang — Portfolio Website

This repository contains the source code for my personal portfolio website.  
The site showcases selected work in **Product Design, Engineering, and AI exploration**, including projects from my professional experience and graduate studies.

🌐 **Live Site:** https://www.jieyangdesign.com  
📧 **Contact:** jieydesign@gmail.com  
💼 **LinkedIn:** https://linkedin.com/in/jieyangdesign  

---

## Overview

This portfolio highlights work across design, coding, and emerging technologies.  
Projects include product design case studies, AI experiments, and software projects built during my studies in computer science.

The goal of this site is to present both **design thinking and technical capability** in one place.

---

## Tech Stack

- **Framework:** Next.js
- **Language:** TypeScript / JavaScript
- **UI:** React
- **Styling:** CSS / Tailwind / custom styles
- **Hosting:** Vercel
- **Version Control:** Git + GitHub
